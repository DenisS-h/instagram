# Proyecto Métricas de Calidad del Software

## Descripción

Este proyecto es parte del curso de Métricas de Calidad del Software del Cuarto Semestre de la Carrera en Desarrollo de Software.

## Estructura del Proyecto

- `app.py` - Aplicación principal
- `send_phish.py` - Script auxiliar
- `templates/` - Plantillas HTML
- `style.css` - Estilos
- `script.js` - Scripts del cliente
- `requirements.txt` - Dependencias de Python

## Requisitos

- Python 3.x
- Flask (u otro framework web)

## Instalación

```bash
pip install -r requirements.txt
```

## Uso

Para ejecutar la aplicación:

```bash
python app.py
```
